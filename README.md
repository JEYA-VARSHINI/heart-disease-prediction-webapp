# ❤️ Heart Disease Prediction Web Application

## 📌 Project Overview

The Heart Disease Prediction Web Application is a Machine Learning-based healthcare project developed using Python, Flask, and Scikit-learn.

The application predicts whether a patient is at risk of heart disease based on clinical parameters entered by the user. It provides an interactive dashboard displaying the prediction result, confidence score, patient details, and health recommendations.

---

## 🎯 Objectives

- Predict the risk of heart disease using Machine Learning.
- Build an interactive web application using Flask.
- Display prediction confidence and patient details.
- Provide basic health recommendations.
- Create a responsive and user-friendly dashboard.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/JEYA-VARSHINI/heart-disease-prediction.git
```

Move into the project folder

```bash
cd heart-disease-prediction
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the virtual environment

### Windows

```bash
venv\Scripts\activate
```

Install required packages

```bash
pip install -r requirements.txt
```

Run the Flask application

```bash
python app.py
```

Open the browser

```
http://127.0.0.1:5000
```
